# ⚖️ Automação PJe 

Este projeto simula uma automação real feita para o sistema **PJe TRT2 e TRF3**, utilizando **Node.js** e **Puppeteer**.  
A automação realiza login com cpf e senha, acessa um processo e anexa uma petição de forma automatizada.

> 🔒 Este repositório é uma versão genérica da automação usada profissionalmente, com informações confidenciais removidas.

---

## 🚀 Funcionalidades

- Login automatizado via cpf e senha
- Preenchimento de número de processo
- Seleção de tipo de petição
- Upload automático de arquivo PDF
- Assinatura digital e envio

---
